PyOpenGL

安装pyOpenGL

  • 默认已经安装了Python3.4
  • 一般的安装方式就是使用 pip install PyOpenGL PyOpenGL_accelerate 直接安装PyOpenGL和PyOpenGL_accelerate

解决问题

在这个过程中可能会出现这样几个问题：

  1. 需要对应版本的 Visual Studio,直接安装解决问题。下载地址

  2. 安装成功后，运行实例代码出现如下错误提示

    C:\Users\Greg\Downloads>shader_sphere.py
    WARNING:OpenGL.arrays.numpymodule:Unable to load numpy_formathandler accelerator from OpenGL_accelerate
    Traceback (most recent call last):
    File “C:\Users\Greg\Downloads\shader_sphere.py”, line 116, in
    TestContext.ContextMainLoop()
    File “C:\Python27\lib\site-packages\openglcontext-2.1.0a11-py2.7.egg\OpenGLContext\glutcontext.py”, line 154, in ContextMainLoop
    render = cls( args, *named)
    File “C:\Python27\lib\site-packages\openglcontext-2.1.0a11-py2.7.egg\OpenGLContext\glutcontext.py”, line 30, in init
    glutInitDisplayMode( self.DISPLAYMODE )

    这是因为下载的是32位的原因，而你需要下载64位的下载链接

    python下如何安装.whl包？
    下载.whl包
    先pip install wheel
    之后pip install 包名字.whl 即可安装某模块包

  3. 报错如下：

    glutCreateWindow( “cube” )
    ctypes.ArgumentError: argument 1: : wrong type

    解决方案

    glutCreateWindow( b”cube” )
