第一次笔试总结

一、编程题

问题:

时间限制：1秒
空间限制：32768K
度度熊想去商场买一顶帽子，商场里有N顶帽子，有些帽子的价格可能相同。度度熊想买一顶价格第三便宜的帽子，问第三便宜的帽子价格是多少？
输入描述:

首先输入一个正整数N（N <= 50），接下来输入N个数表示每顶帽子的价格（价格均是正整数，且小于等于1000）

输出描述:

如果存在第三便宜的帽子，请输出这个价格是多少，否则输出-1

输入例子:

10
10 10 10 10 20 20 30 30 40 40

输出例子:

30

后来晚上写出来的答案：

var readline = require('readline');

var  rl = readline.createInterface({
    input:process.stdin,
    output:process.stdout
});
//上面是输入输出实例
var arr,N=-1;

rl.on('line', function(line){
    if (N < 0) { // 第一行读取n
        N = parseInt(line.trim())
    }else{
        var arr = line.split(' ');
        if(arr.length != N){
            console.log(-1);
        }else{
            arr.sort(function(a,b){
                return a-b;
            });
            var newarr = [],k = 1;
            newarr[0] = arr [0];
            for(var i = 1; i < arr.length; i++){
                if(arr[i] != arr[i-1]){
                    newarr.push(arr[i]);
                    k++;
                }
                if(k==3) {
                    console.log(newarr)
                    return;
                };
            }
            console.log(-1)
            return;
        }  
    }    
});

先说编程上的问题：

  1. sort()对 1 100 2 3 23 直接排序出来的结果是 1 100 2 23 3，原来没注意这个问题这里应该用arr.sort(function(a,b){return a-b;});。

  2. 没怎么编过这种直接编的题，一开始直接蒙了，输入输出查了会，没有经验，自己在那用编辑器调试了好久，主要就是第一个编程题，自己也没那么专注，浪费时间。心里居然还在纠结我查网页会不会出问题。

  3. 编程上有这样几个问题，①算法和数据结构 ②node ③基础代码不够 ④遇见不会的不要慌也不要慢悠悠 ⑤做题先看看难易，由简到难好好做
